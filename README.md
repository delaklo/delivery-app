# Food Delivery App

This is my implementation of a food delivery application. I aimed for an **Advanced Level** solution.

Live: https://delivery-app-green-nine.vercel.app

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express, TypeScript
- **Database:** MongoDB, Mongoose

## Main Features

### Base
- Restaurant & product browsing from database
- Shopping cart with quantity controls
- Form validation 

### Middle
- Responsive design 
- Product filtering by category
- Product sorting 
- Restaurant filtering by rating

### Advanced
- Pagination 
- Order history with search
- Quick reorder functionality

## Quick Start

```bash

cd backend && npm install && npm run seed && npm run dev

cd frontend && npm install && npm run dev
```
